Grayscale as luminance in sRGB color space

https://en.wikipedia.org/wiki/SRGB
Formula (see https://en.wikipedia.org/wiki/Grayscale):
R'G'B' (sRGB,nonlinear) => RGB (linear) => Y (linear) => Y' (sRGB,nonlinear)
Y = 0.2126*R+0.7152*G+0.0722*B

https://en.wikipedia.org/wiki/Rec._601
Formula:
R'G'B' (BT.601) => Y' (BT.601)
Y' = 0.299*R'+0.587*G'+0.114*B'

Grayscale as intensity in HSI model

https://en.wikipedia.org/wiki/HSL_and_HSV
Formula:
RGB => I (HSI)
I = avg(R,G,B) = (R+G+B)/3




Grayscale as lightness in HSL model

https://en.wikipedia.org/wiki/HSL_and_HSV
Formula:
RGB => L (HSL)
L = mid(R,G,B) = (max(R,G,B)+min(R,G,B))/2




Grayscale as value in HSV model

https://en.wikipedia.org/wiki/HSL_and_HSV
Formula:
RGB => V (HSV)
V = max(R,G,B)
